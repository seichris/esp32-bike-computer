that controls, is controlled by, or is under common control with that entity.
"Control" means ownership of at least fifty percent of the voting interests or
the power to direct the entity's management.

You accept this Agreement by selecting the CLA acknowledgement in the pull
request template and submitting the pull request. You adopt the GitHub account
used to submit that acknowledgement as Your electronic signature. If You are
acting for a legal entity, identify that entity in the pull request.

This Agreement takes effect when You first accept it. It covers the
Contributions identified by that pull request and future Contributions You
submit using the same GitHub account unless We agree otherwise in writing.

"Contribution" means any original work of authorship that You intentionally
submit to Us for inclusion in the repository and in which You own or control
copyright. It includes modifications and additions to existing material. It
does not include third-party material in which You do not own or control

control system, or another channel We manage for improving the project. A

"Material" means the Open Bike Computer component to which You submit a
Contribution. "Submission Date" means the date You submit that Contribution.
